### üèÅ Getting Started

#### Prerequisites
- Node.js (v18+)
- npm or bun

#### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd patria-nexus
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Environment Setup:
   Create a `.env` file (or configure via UI):
   ```env
   VITE_OPENAI_API_KEY=your_api_key_here
   ```

4. Launch Development Server:
   ```bash
   npm run dev
   ```
